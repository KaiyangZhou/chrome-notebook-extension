# Chrome Notes

A Chrome extension for writing notes in a side panel.

## Install

1. Open `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this folder

Optional: Add PNG icons (16, 48, 128) in `icons/` and add an `icons` key to `manifest.json` for a custom toolbar icon.

## Usage

Click the extension icon to open the notes side panel directly. Add, edit, and delete notes; data is saved locally.
